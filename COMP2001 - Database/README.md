# COMP2001 JBott

## App Features

The micro-service contains the following:
* Login Using The Existing Authenicator API
  * https://web.socem.plymouth.ac.uk/COMP2001/auth/api/user
* View All Profiles
* View Users Profiles
* Create New Profile
* Edit Existing Profile
* Delete Profile
  * Admin Only
  * Archives Not Deletes
